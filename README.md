# Museum of Thought (متحف الفكر)

A digital gateway for intellectual creativity, sharing ideas, philosophy, and knowledge.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env` (refer to `.env.example`).
4. Run the development server: `npm run dev`

## Features

- Ideas Gallery
- Commenting and Replying
- User Profiles
- Admin Dashboard
- Authentication
